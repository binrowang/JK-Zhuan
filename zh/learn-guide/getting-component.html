# 组件化

本文讲解即刻赚项目中主要使用到组件

# top-bar

顶部栏

# 参数

参数名 介绍 必选 值类型 可选值 默认值
title 标题 非 String
routerLink 指定路由跳转 非 String 返回上级路由

# 使用方法

由于复用性高，在global/components/**.vue 进行全局的组件注册，可直接调用组件

提示

routerLink 跳转的路由的name属性

    <template>
        <div>
            <top-bar title="即刻赚公告" routerLink="personalCenter"></top-bar>
        </div>
    </template>

# loading

加载动画

# 参数

参数名 介绍 必选 值类型 默认值
isShow 控制是否显示 否 Boolean true

# 使用方法

由于复用性高，在global/components/**.vue 进行全局的组件注册，可直接调用组件

    <template>
        <div> // 通过定义showLoading变量来控制加载动画，实现如任务列表页拉取多个接口平缓的等待
            <loading :is-show="showLoading"></loading>
        </div>
    </template>

同时在app.vue入口写入全局的过渡动画

// --- App.vue
<tempalte>
    <loading v-if="isLoading"></loading> 
</tempalte>
<script> // 通过全局状态管理的进行控制
import {mapState} from "vuex"
export default {
    computed:{
      ...mapState([
          'isLoading'
        ])
    }
}
</script>

通过路由的跳转来通知vuex

// --- main.js 
router.beforeEach((to, from, next) => {
  store.dispatch("onLoading",true);
  next()
})
// 这里为了让效果明显一些加了延时
router.afterEach((to, from) => {
  setTimeout(() => {    
    store.dispatch("onLoading",false); 
  }, 0);
})

// --- store/index.js
	state: {
		isLoading: false,
	},
	mutations: {
		updateLoadingState(state, flag){
			state.isLoading = flag
		},
	},
	actions: {
		onLoading(context, flag){
			context.commit("updateLoadingState", flag);
		}
	}

# jk-dialog

模态框

# 参数

参数名 介绍 必选 值类型 可选值 默认值
isShow 控制是否显示 非 Boolean false
title 标题 非 String 温馨提示
isClose 关闭弹窗 非 Boolean true

由于复用性高，在global/components/**.vue 进行全局的组件注册,可直接调用组件,可插槽

<template>
    <!-- 助手开启 -->
    <jk-dialog :is-show="isHelperOpen" @is-close="isHelperOpen=false">
        <div class="register-tip-info">
            <p>哎呀!任务助手未打开</p>
        </div>
        <div @click="openHelperUrl" class="continue-tip">
            打开助手
        </div>
        <h4 class="install-helper">如您未安装助手,请
            <span @click="downloadOpen" class="linkHelper">下载助手</span>
        </h4>
    </jk-dialog>
</template>

# download-helper

下载助手

# 参数

参数名 介绍 必选 值类型 可选值 默认值
downloadURL 下载地址 否 String 无
downloadImg 图片 否 String 无
appTitle 产品名字 否 String 无

提示

download-helper 的参数通过调用接口获取

import {mapActions} from 'vuex'
...mapActions('assistant', ['ajaxAppIntroduce'])
let res = await this.ajaxAppIntroduce()

由于复用性高，在global/components/**.vue 进行全局的组件注册,可直接调用组件

<!-- 下载助手 -->
<template>
    <download-helper :downloadURL = urlScheme :downloadImg = downImg  :appTitle = urlTitle></download-helper>
</template>

# yd-icon

VUE YDUI 第三方UI组件库提供

# 参数

参数名 介绍 类型 默认值
name 图标名称(必填) String
color 图标颜色 String && 常规颜色值 #333
size 图标大小 String && (px或ren) .6rem(30px)
custom 是否为自定义图标 Boolean false

# 使用方法

第三方提供的图标

// 直接使用name引入即可, size 和 color 为可选值
<yd-icon name="feedback" size="0.5rem" color="#cccccc"></yd-icon>

自定义图标

提示

iconfont下载图标将**.ttf文件放到src/common/iconfont/ 目录下

自定义字体图标在src/common/stylus/base.styl 目录下定义

/* 自定义字体图标 */
@font-face 
    font-family 'font_family'
    src url('../iconfont/iconfont.ttf') format('truetype')

[class^="icon-custom-"]:before, [class*=" icon-custom-"]:before 
    font-family 'font_family'

.icon-custom-pengyouquan:before 
    content "\e73b"
    ...
    ...

使用自己定义的字体图标

// name为自定义图标的名字，同时要加上custom来识别区分
<yd-icon size="0.5rem" name="pengyouquan" color="#d8d8d8" custom></yd-icon>

# dialog

常见移动端弹窗提示框集合类，提供confim、alert、toast、loading、notify五个组件

# 参数

参数名 介绍 类型 可选值 默认值
mes 提示信息 String 无
icon 图标类型 String success，error，none none
timeout 显示时长（单位毫米） Number 2000
callback 回调函数 Function 无

# 使用方法

this.$dialog.toast({
    mes: '发送中...',
    timeout: 1500
});

# 效果

# yd-accordion

折叠面板,移动端常见将内容展开或折叠的组件。

# yd-accordion 参数

参数名 介绍 类型 默认值
accordion 是否可以同时展开多个面板 Boolean false

# yd-accordion-item 参数

参数名 介绍 类型 默认值
title 标题 String 无
open 是否默认展开面板 Boolean false
auto 是否自动展开折叠参数 Boolean true

# 使用方法

<template>
    // 可参考src/page/user/tryIncome.vue
    <yd-accordion accordion>
        <yd-accordion-item class="try-income" v-for="(item,index) in income" :title="index" :key="index" open>
            <div class="try-income-info">
                <ul>
                    <li v-for="(itemSon,index) in item" :key="index">
                        <div class="try-income-icon">
                            <img v-if="itemSon.type===2" :src="itemSon.image" alt="">
                        </div>
                    </li>
                </ul>
            </div>
        </yd-accordion-item>
    </yd-accordion>
</template>
