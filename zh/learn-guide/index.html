# 介绍

即刻赚 是一款移动端积分墙推广项目,由深圳市乐瀛文化科技有限公司推出的一款全新的赚钱类应用，玩手机就能轻松赚钱，提供多种获取收益方式：

  • 完成每日任务即可获得奖励
  • 邀请好友各获得奖励，和好友一起赚钱更好玩只需关注微信就可快速提现
  • 我们致力于提高投放移动广告效果，优化用户体验，为用户带来更多收益。快来和我们一起即刻赚吧。

TIP

即刻赚网址：http://jkzhuan.com/

提示：请切换到移动端打开

# 功能

  • 使用 vue-cli2 构建
  • vue-ydui UI组件库
  • axios请求
  • lib-flexible + px2rem-loader rem布局
  • clipboard 复制板
  • stylus 预处理语言
  • aes.js aes解密
  • vconsole.min.js 移动端控制台
  • vue-qriously 邀请二维码
  • fastclick 触摸延迟
  • iconfont 图标库 用户名(sixgay)

# 目录结构

├─ .vscode 部署环境
├─ build webpack配置
├─ config 配置
├─ docs 文档
├─ node_modules 项目依赖
├─ src
│  ├─ assets 图片资源
│  ├─ common 字体图标|样式

│  ├─ global 全局
│  ├─ page 页面
│  ├─ port 接口(已弃)

│  ├─ util 公共方法
│  ├─ store 全局状态 | 接口管理
│  ├─ App.vue
│  ├─ main.js
├─ statics 静态资源
├─ test 单元测试
├─ .gitignore
├─ .postcssrc.js
├─ LICENSE
├─ README.md
├─ package.json
